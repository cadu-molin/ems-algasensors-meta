# ems-algasensors-meta

## 📥 Clonando o repositório com submódulos

Para clonar este projeto juntamente com os submódulos, execute o comando:

```bash
git clone --recurse-submodules -j8 [nome-do-repositorio-principal]
```

## 🔄 Atualizando os submódulos

Para atualizar os submódulos após alterações, utilize:

```bash
git pull --recurse-submodules
```
